# 📱 StageFlow Mobile App

---

## 👩‍💻 About the App
**StageFlow Mobile** is a cross-platform mobile application that helps students manage internships easily.  
It connects students with companies and allows them to apply, track, and manage internship opportunities directly from their phone.

---

## ⚙️ Features
- 👨‍🎓 Student authentication (Login / Register)
- 📄 Browse internship offers
- 📨 Apply to internships easily
- 📊 Track application status
- 🔔 Notifications (if implemented)
- 📱 Mobile-friendly UI (Android / iOS)

---

## 🛠️ Tech Stack
- React Native (or Expo)
- JavaScript
- Node.js (API backend)
- Express.js
- MongoDB
- JWT Authentication
